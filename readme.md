# Features
- Auto install
- Runs on Startup
- Works with X86 and Appdata Roblox Installs

# Custom audio system
- to use, download your file and convert it to .ogg
- name the file ```useme.ogg```
- place it on your documents
- run Update.bat
- if your file isn't found, the default (OOF) will be downloaded.
